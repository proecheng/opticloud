# 案例 1 — 🚚 物流：同城闪送路径优化

## 客户场景

**「快盛同城配」** 是一家 200 人规模的同城配送公司，每天接 **3,000-5,000 单**外卖、文件、鲜花等急送需求，覆盖某二线城市主城区。

- 50 辆电动车（早班 30 / 晚班 20 轮换）
- 30 个分布全城的中转点
- 客户期望：**1 小时内送达**

## 痛点

1. **排路线靠老员工经验** — 一个老调度员每天上午要花 **3 小时** 在地图上画路线
2. **超时率 8%** — 每月约 **¥40 万违约金**
3. **空驶率 25%** — 司机往返中转点空跑，每月油 / 电费多花 **¥20 万**

# 案例 2 — 🚛 物流：跨城货运 + 顺路返程匹配

## 客户场景

**「华运通」** 是一家以鲁苏皖三省为主的中型公路货运公司，自有车队 **120 辆重卡**，主营干线运输（杭州 → 武汉、青岛 → 西安等 30 条主线）。

- **每月发车 8,000 次**
- **返程空驶率 38%**（卡车从目的地空跑回起点）
- 一辆重卡空驶一次 = 烧掉 **¥4,000 油费 + 司机工资**

## 痛点

1. **38% 返程空驶 = 每月直接亏 ¥1,200 万**（8,000 × 38% × ¥4,000）
2. **找返程货全靠司机自己在路边停车场打听** — 老旧、信息不对称
3. **车队调度员 5 人 × 24 小时三班倒** — 月人力 **¥80,000**
4. **货物 / 卡车类型不匹配** — 司机硬塞货物导致车损率上升

## 客户对话剧本

> 👤 **客户（运营总监李总）**：我们最大成本就是返程空跑。市面有"货拉拉"、"运满满"这种平台，但都是 C2C 散单，我们是 B2B 整车，对接不上。

> 👩 **OptiCloud 顾问**：李总，这是个**双向匹配优化问题** —— 一辆 9.6 米厢车从杭州到武汉，明天上午到，能拉 25 吨，怎么在武汉就近找到一票货回杭州或附近？

> 👤 **李总**：你们能接外部货源平台？

# 案例 3 — 🏥 医疗：三甲医院手术室排班优化

## 客户场景

**「市第一人民医院」** 是 1,500 床位的三甲综合医院，**12 间手术室 / 50 名外科医生 / 25 名麻醉师**，每天做 **80-120 台手术**。

约束极其复杂：
- 主刀医生只做特定专科（心外、骨科、神经外科…）
- 麻醉师有不同等级（一级 / 二级 / 三级）
- 急诊手术随时插入
- 术后 ICU 床位 / 普通病房床位 / 康复床位都有上限
- 高级仪器（DSA / 达芬奇）共享

## 痛点

1. **手术室周转时间长**：每台手术间隔均 **45 分钟**（行业平均 25 分钟）
2. **医生抱怨手术堆叠 / 待机** — 王主任今天连做 6 台心外科手术后又被排了 4 台胆囊
3. **病房床位频繁阻塞** — 术后没有 ICU 床位 → 手术延期 → 患者投诉
4. **现状人工排班**：护士长每周日加班 8 小时排下周班表，每周还有 5-10 次临时调整
5. **行政压力**：医院 KPI 要求**手术量提升 20%**，但物理资源没增加

## 客户对话剧本

>
> 👤 **李处长**：我们行政信息科只有 3 个人，搞不定算法的事。
>

# 案例 4 — 💊 医疗：县医院药品库存预测

## 客户场景

**「鲁西县人民医院」** 是 400 床位的县级二甲医院，药剂科库存 **800+ SKU**（西药 + 中成药 + 部分生物制品 + 冷链疫苗）。

- 月营业额：药品销售约 **¥600 万**
- 药剂科 8 名药师 + 1 名采购员
- 上游：3 家主供药厂 + 5 家本地代理商

## 痛点

1. **缺货被投诉** — 经常患者来开药「这个药没有」，月均 **30 单投诉**，医保 KPI 扣分
2. **过期损耗** — 月均 **¥18 万** 药品过期或近效期处理掉
3. **采购员凭"感觉"补货** — 看上月销量 + 微信群消息，特别是**冷链药品** 经常断货或积压
4. **季节波动**：流感季 / 高考前感冒药 / 冬季腹泻药 / 慢性病淡旺季 — 总在事后才反应过来
5. **效期管理**：先入先出 vs 后入先出搞错 → 损耗

## 客户对话剧本

> 👤 **客户（药剂科张主任）**：我们采购员老李 5 年了，他凭经验干得还行，但他要退休。我们没法养"数据科学家"，怎么办？
>

>
> 👤 **张主任**：能预测准吗？县医院数据量小。
>
> 👩：好问题。我们用**全球开源时序基础模型**（Chronos / TimesFM / Lag-Llama），它们在数百万真实时间序列上预训练，**少样本场景准确度高于本地训练模型**。你只需提供过去 12-24 个月销量数据，**每周自动 retrain**。
>
